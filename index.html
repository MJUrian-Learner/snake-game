SNAKE COLOR

DID YOU KNOW?

Although popularized by Nokia, the Snake Game was first created in 1976 by Gremlin Interactive, a game development company, as a video game named Blockage. It was supposed to be a black-and-white arcade game meant to be played by two players.

HOW TO PLAY?

Use the arrow keys to move the snake and eat the food to grow longer. Avoid eating yourself!

Make use of the following features for a more challenging gameplay:

WALL - You cannot touch the walls anymore, doing so means game over for you.

ROCKS - Spawns a rocky terrain, watch your head!
